# SIGA Aduanas - Prototipo v0.1.0

Versión inicial del prototipo funcional SIGA para el Paso Los Libertadores.

## Alcance de esta versión

Esta versión representa el primer avance del proyecto:

- pantalla de inicio de sesión;
- panel principal básico;
- consulta de trámites;
- creación de pre-trámite;
- datos simulados en navegador.

## Módulos aún no implementados en esta versión

En esta primera versión todavía no se incorporan los módulos completos de menores de edad, mascotas, SAG detallado, PDI avanzado, conciliación, flujo de ventanillas ni validaciones finales.

## Técnica de prototipado

Prototipo web estático desarrollado con HTML, CSS y JavaScript.
